🗳️ Hashgreen 去中心化自治組織

HashgreenSwap 以成為真正的去中心化交易所為目標發展，期望建立一個安全且權力分散化的去中心化交易所。所以在未來 HashgreenSwap 的治理將會交給 HashgreenDAO ，而 HG token 會是參與 DAO 的憑證。

HG token 主要由年化農場發放，所有人皆能透過提供質押 LP 代幣的方式來獲取 HG token，這保證利益是分散的，且將治理的權力分散給所有在乎 HashgreenSwap 發展的大家。

HashgreenDAO 協議​

共享手續費​

我們鼓勵所有持有 HG token 的用戶，長期參與 HashgreenDAO 的治理。我們提供了 HG token 的“質押池”，若是你將 HG token 存入其中，你將與所有質押 HG token 的持有者一起享有 HashgreenSwap 交易手續費的0.05%的返利，返利將會以 HG token 的形式返還至質押者手中，歡迎你質押你手中的 HG token 來獲得返利資格。

投票​

雖然我們期望能夠儘早地達成去中心化，但是過早的去中心化治理是不明智的，這會使得組織面臨龐大的風險，在未來，隨著 HG token 的發放，我們會逐漸將 HashgreenSwap 的經營轉移給社群，當團隊確定社群發展足夠健全，能夠有足夠韌性面對不可預知的風險時，我們將完全轉交權利，開啟投票機制，讓每個 HashgreenDAO 的參與者，皆能夠透過投票的方式，參與 HashgreenSwap 的建設，成為一個真正的去中心化自治組織。

未來規劃​

在未來我們將陸續建立更多的 DeFi 協議，增添 HashgreenDAO 的多元性，我們期望 HashgreenDAO 能夠成為完全自主且具備發展潛力的組織，但這不是件容易的事情，也不是匆忙情況下能夠完成的，我們歡迎所有人與我們一同參與我們的社群討論 HashgreenDAO 未來該如何運作和發展。
